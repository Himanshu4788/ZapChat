# ZapChat
 Zapchat is a multi-threaded real-time chat application that enables multiple users to communicate concurrently. The project is designed with scalability and performance in mind, using efficient threading and asynchronous communication techniques. It demonstrates core concepts such as distributed systems, concurrency, synchronization, and fault-tolerant design.

This project is suitable for understanding real-time systems, system design fundamentals, and modern chat application architecture.
